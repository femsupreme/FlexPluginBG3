
# FlexPluginBG3

A plugin to allow Baldur&#x27;s Gate 3 info to be displayed on the FlexBar.

## Installation


### **Prerequisites**

- Node.js 18 or later  
- FlexDesigner v1.0.0 or later  
- A Flexbar device 
- Install FlexCLI  
  ```
  npm install -g @eniac/flexcli
  ```

### Clone & Setup

```
git clone https://github.com/femsupreme/FlexPluginBG3.git
cd FlexPluginBG3
npm install
```

## Debug

```
npm run dev
```

## Build & Pack

```
npm run build
npm run plugin:pack --path com.eniac.example.plugin
```
